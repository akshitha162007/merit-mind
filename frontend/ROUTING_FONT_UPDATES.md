## Font Styling Changes

### Previous Fonts
- Syne (headings) - Too bold/display-focused
- DM Sans (body) - Generic

### New Professional Fonts
- **Poppins** (headings) - Modern, professional, clean
  - Weight: 600, 700, 800
  - Better for corporate/professional look
  
- **Inter** (body) - Industry standard for web
  - Weight: 300, 400, 500, 600, 700, 800
  - Excellent readability
  - Professional appearance

### Typography Hierarchy
```
h1: 3.5rem, weight 800, letter-spacing -0.5px
h2: 2.5rem, weight 700, letter-spacing -0.5px
h3: 1.5rem, weight 700, letter-spacing -0.5px
p: 1rem, weight 400, line-height 1.7, letter-spacing 0.3px
```
